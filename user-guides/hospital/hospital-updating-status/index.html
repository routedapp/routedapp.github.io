Updating Hospital Status

  1. Click the Update Hospital Info button to edit the number of Available Beds or the ER conditions.

  2. Click the + or - buttons next to the bed counts to increase or decrease the bed counts, respectively. Type any ER conditions notes in the text box. Click Confirm Updates to make these changes visible to EMS teams.
